# Signature Component

This is a signature component designed to be easily added to any web page. It serves as an authentication mark to signify that the content on the page was created by `@brandoniscoding`.

## Features

- **Fixed Position**: The signature remains fixed at the bottom of the viewport, ensuring it is always visible regardless of page scroll.
- **Animations**: The text has a reveal animation from left to right, an oscillation effect, and synchronized color changes from white to black.
- **Hover and Click Effects**: Enhanced interactivity with effects when hovering over and clicking the text.
- **Minimalist Design**: No impact on the existing page content, designed to blend in with various designs.

## Usage

To use this signature component, include the HTML and CSS provided in the repository in your project.
